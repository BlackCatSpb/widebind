# WideBind для занятых

Языковая модель без transformer-слоёв. Нет attention, нет softmax, нет KV-cache.

## Три идеи

**1. Память — вектор, не матрица.** Один D-мерный вектор на слой (суперпозиция), не матрица K×V всех прошлых токенов. ~16 KB на слой — генерация хоть миллион токенов без роста памяти.

**2. Скрещивание размерностей — биллинейность, не weighted sum.** Проекция D→K=64, покомпонентное произведение u⊙v, проекция K→D. Ранг M≤64, но все D² элемента M соединены.

**3. Ни softmax, ни attention.** Sigmoid только для VSA-гейтов. Tanh только для Mirror. Всё остальное — линейные проекции + SiLU.

## Ключевые числа

- **161M params**, 32 слоя, D=4096
- **32 группы** всё: MLP, Mirror, эмбеддинг, голова
- **K=64 bottleneck**, Фибоначчи-твист (off/shift/cascade)
- **87.9% параметров** — GroupedMLP
- **Эмбеддинг + голова**: 8192 параметра (0.01%)
- **VSA scan** — O(L log L), chunked, численно стабилен

Чекпоинт: 159 MB (FCF-CPR, 8-bit). Инференс: ~0.55 GB VRAM (fp16).
